The Last Hotel

Jack Fortescue

Introduction

The Last Hotel is a First Person Virtual Reality Horror game made in Unreal Engine 4. It was developed by myself and two of my coursemates. I was Project Manager and Lead Programmer on this game.


Inspired by "Slender", we created a horror experience set in an old abandoned hotel. You are tasked with collecting five notes before escaping with your life.

The game was designed to be played using the Oculus Rift

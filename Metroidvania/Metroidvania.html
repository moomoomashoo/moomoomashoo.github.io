Prototype Metroidvania

Introduction

This game is a short, rough prototype of a Metroid-style 2D shooter/platformer

It simply showcases a few gameplay elements that I wanted to play around with. This includes; breakable blocks that respawn, an gun aimed using the mouse cursor, underwater movement, slippery ice, and the ability to switch between vision modes.
